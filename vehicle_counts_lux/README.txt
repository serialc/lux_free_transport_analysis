# Instructions

- Download the vehicle count data from [data.public.lu](https://data.public.lu/en/datasets/pch-comptage-trafic/#resource-community-87e907df-de72-4478-a872-7e4f4048835c)
- Move the data to the vehicle_counts_data directory
- Run ```libreoffice --headless --convert-to csv *.xlsx``` on the xlsx files to convert them to csv
- Run the python script ```parse_counts.py```
